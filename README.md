# Pocket dimensions for Paper

## Build

Get the source from GitHub and build it with gradle:

```shell
git clone https://github.com/IPOleksenko/PocketHome.git
cd PocketHome/
./gradlew build
```

### Development server

__After building__, you can start a development server:

#### Windows:

```powershell
run/start.ps1
```

#### Linux:

```shell
run/start.sh
```
